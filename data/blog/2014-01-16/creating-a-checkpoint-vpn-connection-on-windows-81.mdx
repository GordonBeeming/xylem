So we use [CheckPoint VPN](http://www.checkpoint.com/resources/virtual-private-network/), but the most frustrating part is that when a new version on windows is released the CheckPoint VPN client is incompatible with it and makes your computer weird when you do try use it.
 
For a while now I (and others) have been waiting for a client for windows 8.1. Very recently I found out that this functionality is baked into Windows 8.1 already and we don't have to wait ![Smile](https://gordonbeeming.com/images/emoticons/smile.svg). Here's how to create the VPN connection
 
Open the Charms bar (slide from the right or Win + C) and click Settings
 
<Figure key="/images/fe4ec651735542c0a2ff1ee8b7377a4a.png" src="/images/fe4ec651735542c0a2ff1ee8b7377a4a.png" alt="step 1" width="0" height="0" caption="FILL THIS IN" />
 
Select Change PC Settings
 
<Figure key="/images/de9b325f3a324275a3c7273f4c05f225.png" src="/images/de9b325f3a324275a3c7273f4c05f225.png" alt="step 2" width="0" height="0" caption="FILL THIS IN" />
 
Click on Network
 
<Figure key="/images/79b33e4090b747c09dddae6cf48f1dc7.png" src="/images/79b33e4090b747c09dddae6cf48f1dc7.png" alt="step 3" width="0" height="0" caption="FILL THIS IN" />
 
Under connections select Add a VPN connection
 
<Figure key="/images/53c74d53fcd14b5c8086a0a55f12e277.png" src="/images/53c74d53fcd14b5c8086a0a55f12e277.png" alt="step 4" width="0" height="0" caption="FILL THIS IN" />
 
Choose Check Point VPN as the provider, enter a name for the connection and also enter the server name or address and click Save
 
<Figure key="/images/5f09ae6934734adf82ff2164f2e4b466.png" src="/images/5f09ae6934734adf82ff2164f2e4b466.png" alt="step 5" width="0" height="0" caption="FILL THIS IN" />
 
You should see your connection in the Connections page now
 
<Figure key="/images/3ed9d59ae7fe481da296e14d03938297.png" src="/images/3ed9d59ae7fe481da296e14d03938297.png" alt="step 6" width="0" height="0" caption="FILL THIS IN" />
 
Head back to the charms bar and click Networks
 
<Figure key="/images/ca0b1df176da474ca471fe9da044781f.png" src="/images/ca0b1df176da474ca471fe9da044781f.png" alt="step 7" width="0" height="0" caption="FILL THIS IN" />
 
Under Connections you will see your VPN connection, Click on it
 
<Figure key="/images/d6d7191528e74bb5b42c726fe1a17e23.png" src="/images/d6d7191528e74bb5b42c726fe1a17e23.png" alt="step 8" width="0" height="0" caption="FILL THIS IN" />
 
Click on Connect
 
<Figure key="/images/4e13f590f4d44d66add657d627d47f55.png" src="/images/4e13f590f4d44d66add657d627d47f55.png" alt="step 9" width="0" height="0" caption="FILL THIS IN" />
 
Select the authentication method (I'll use Username and Password), Click Next
 
<Figure key="/images/e622f45321ea48109d264e9811062c0a.png" src="/images/e622f45321ea48109d264e9811062c0a.png" alt="step 10" width="0" height="0" caption="FILL THIS IN" />
 
Enter the Username and Password and click OK
 
<Figure key="/images/853d2361560549e9925f481f0c58a029.png" src="/images/853d2361560549e9925f481f0c58a029.png" alt="step 11" width="0" height="0" caption="FILL THIS IN" />
 
You will now be connected
 
<Figure key="/images/ca07bd41539f48dca2a837b8ae944ed0.png" src="/images/ca07bd41539f48dca2a837b8ae944ed0.png" alt="step 12" width="0" height="0" caption="FILL THIS IN" />
 
It's that simple, no more waiting around for VPN providers to update their clients ![Smile](https://gordonbeeming.com/images/emoticons/smile.svg)
