[Visual Studio](http://www.visualstudio.com/) 2013 Update 1 has been released and as usual you are able to use the /layout argument to download all the files so you can share them locally instead of everyone downloading them. As mentioned in [Brian Harry](http://blogs.msdn.com/b/bharry/)'s blog [here](http://blogs.msdn.com/b/bharry/archive/2014/01/20/vs-2013-1-update-1-is-available.aspx) there is no update for TFS this time round but good things are coming for TFS Update 2.
 
If you don't know how keep reading ![Smile](https://gordonbeeming.com/images/emoticons/smile.svg).
 
Download the update file from [http://www.microsoft.com/en-us/download/details.aspx?id=41650](http://www.microsoft.com/en-us/download/details.aspx?id=41650 "http://www.microsoft.com/en-us/download/details.aspx?id=41650"). Hold down Shift and Right Click in the folder where you downloaded the file to and click on **Open command window here**.
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/65daa0d3b4b10b252f0777ebea0ed7cb58becdaa/12292478-baf8-4fc5-ba65-f0e476b95197.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/551a00020f8f50471cea666edc44c60ec2e8b573/6c3b6c58-3b49-4130-a38d-761e13a8b23a.png)
 
In the command window type **VS2013.1.exe /layout** and press enter
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/13710955191407b2ffb286477609e388419a40fb/d3699d18-54ba-446c-8077-788ac0a0a700.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/88e307144cf9fb154a71f95df5d74f892694ebf1/3c594594-1ab2-4846-9a03-540bda47e774.png)
 
Setup will run and you will see a different screen from normal
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/5a5e7039ec64f34aec0de86bbb748cb8274bb4c7/6d727591-afbd-4439-8d46-b9691e48d6f2.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/aff0615753ec6cc2f66ff29168765fdda97d8e88/37289a8f-53db-47b4-9f27-b1b3ffd9fc01.png)
 
Select the location you want to save the files to and click DOWNLOAD. The files will all now be downloaded for offline use.
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/f7f3bf8a14a819d2e4b2d8ebf1a6653680ae1902/445ffa11-447e-4956-a113-0bdef64ff354.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/02aaea71bfb43bdcda26cc7158ae086f31648981/bb16fffe-856d-40de-abb1-c328e3d0981a.png)
 
 When the download is complete you can run the VS2013.1.exe executable in the download location 
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/d0855981b834e10f130ab6e7f4ad5773cd3ba6dd/e2954859-607e-4a65-a1f9-398ee2f9fd15.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/3c251a67afa23ea82e0ced5b2286d909ceeb3ec9/b09670c9-b6ea-482a-87c1-823b711de72a.png)
 
The setup will continue as normal but instead of downloading the files the files in the packages location will be used instead. Check the I Agree checkbox and then Install
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/d47f77f38363535b04c22c5ff772c717b1b50a88/4745ee3b-28b1-4bb3-9dd4-90e60f24b9d6.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/5409fec3eb9d31ce9199b7561792309ac1c8e2ce/c65dd4e6-05a7-4769-9af1-86eeeb660d6a.png)
 
As mentioned the packages are now downloaded so after the creating a system restore point you will see the Acquiring bar will go to 100% straight away and the setup will continue installing the components
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/5f0f9c4c6cb90ad2057c5462180b4836dbaae343/4149894b-a9d4-4dbb-8042-3e73a74d8a30.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/cf7c04603aa4742c187424e8d662b4b1961e8b5c/eabd36d3-136a-4d37-8ee7-1ce0ad5528e0.png)
 
When done you might be asked to restart your PC, click Restart Now
 
[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/ccc5e0497e7554a44e907f28863716de98cf760b/413393b4-a289-4df7-ae6b-4c764ddba134.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/778ecd430c7eebfcc8592ca165fe7e0bdfa4632b/7ac3047a-11ae-4b9b-9a7c-d10f3d8ac055.png)
 
Enjoy Update 1 for Visual Studio 2013 ![Smile](https://gordonbeeming.com/images/emoticons/smile.svg)
