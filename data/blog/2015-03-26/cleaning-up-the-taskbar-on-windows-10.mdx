To show that Microsoft is listening below is how you can hide these.
 
## 
 
## Hide Task View button
 
To hide the Task View button simple right click on the Taskbar and then un tick **Show Task View button**.
 
<Figure key="/images/ccf595efc773440ebb2df7e0b399fdb5.png" src="/images/ccf595efc773440ebb2df7e0b399fdb5.png" alt="Hiding Task View button from Taskbar context menu" width="0" height="0" caption="Hiding Task View button from Taskbar context menu" /> 
 
## Hide Search Box
 
Hiding the search box is not much different. Right click on the taskbar and then go to search, in this case you are presented with multiple options. 
 
<Figure key="/images/045808b68bb448d18436f9b31d9071fd.png" src="/images/045808b68bb448d18436f9b31d9071fd.png" alt="Search options in Taskbar context menu" width="0" height="0" caption="Search options in Taskbar context menu" />
