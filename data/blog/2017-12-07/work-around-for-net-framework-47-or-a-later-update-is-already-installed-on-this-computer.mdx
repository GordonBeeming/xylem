I see this message most times I format my machines after opening existing projects in Visual Studio and I always forgot how I solve it

[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/35bfc259249e14ad0d483e62c7e507c707a85361/35bfc259249e14ad0d483e62c7e507c707a85361.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/7a2f73e9d2e28030e544b575560e760e5c2d678b/7a2f73e9d2e28030e544b575560e760e5c2d678b.png)

You may think well that's simple right, install .net 4.7... the only problem then is you get this message which might be why you came to this post ![Smile](https://gordonbeeming.com/images/emoticons/smile.svg)

[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/ca373294faee8200fb30d60188be3c0e4831bbb0/ca373294faee8200fb30d60188be3c0e4831bbb0.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/3f44aac6e5a1055a56cc12c62c87e132f9df0e06/3f44aac6e5a1055a56cc12c62c87e132f9df0e06.png)

The reason for this is that you need to install the Dev Pack for .net 4.7. In my case I already installed .net 4.7.1 and so just installed 4.7 didn't work for me. [https://www.microsoft.com/net/download/windows](https://www.microsoft.com/net/download/windows "https://www.microsoft.com/net/download/windows")
