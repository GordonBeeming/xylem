This is a quick reference post that shows you how to create a branch in VSTS and TFS.

## Steps

The steps for this are quite simple. You will need to

1. Navigate to the Code Hub Group, then
2. Select the Repo you want to set the policy for, then
3. Navigate to the Branches hub, next
4. Click on the ellipse and lastly
5. Click on New branch

[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/ac8db3368728aed8639c98bb697633960bb2a919/ac8db3368728aed8639c98bb697633960bb2a919.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/05ba7ab8be860fc602970ca094d28f62c27267a4/05ba7ab8be860fc602970ca094d28f62c27267a4.png)

Next type in

1. A name for the branch in the folder you would like to set the branch name then
2. Click Create branch

[![image](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/3d1818fa9f977e7ead529fabffc61f935eb8e709/3d1818fa9f977e7ead529fabffc61f935eb8e709.png "image")](https://gordonbeeming.com/blog/binaries/stream/image-jpg/image/1000/0/bbdad8edd00c0708f4d9fcf8069983092d671a89/bbdad8edd00c0708f4d9fcf8069983092d671a89.png)

That’s all and your branch is created
